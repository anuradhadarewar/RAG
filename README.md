🩺 AI Powered Medical RAG Chatbot

📌 Overview
This project is an AI-powered Medical Question Answering Chatbot built using Retrieval-Augmented Generation (RAG). It retrieves relevant information from medical PDF documents and generates accurate answers using Large Language Models.

 🚀 Features
- Medical PDF document processing
- AI-powered question answering
- Retrieval-Augmented Generation (RAG)
- Fast document search
- User-friendly web interface

 🛠 Technologies Used
- Python
- Flask
- LangChain
- FAISS
- Hugging Face
- HTML
- CSS

 📂 Project Structure
- app.py
- store_index.py
- requirements.txt
- setup.py
- Data/
- static/
- templates/

 🎯 Future Improvements
- Multiple PDF support
- Voice input
- Chat history
- Improved UI
